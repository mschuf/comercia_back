# Recursos visuales

- `comercial-repositor-v2.webp`: imagen original generada para Comercia. Muestra
  a un repositor verificando una góndola con tablet y lector, sin marcas ni texto.

El recurso se convirtió a WebP con calidad 88 (1586 × 992, aproximadamente
182 KB). Se importa de manera estática para que Next.js conozca sus dimensiones
y entregue variantes del tamaño adecuado para cada dispositivo.
